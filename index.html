Projects

  • Information Fusion and Multi-dimention Vision on wheeled soccer robot [Slide] [Code]

    I designed a robost framework for our Middle Size League soccer-playing robots, which is majorly composed of omni-vision using GigE paranoma camera and front-vision using ZED RGB-D camera. This work achieved the real-time localization, mapping and object detection and it's done when I'm in wheeled Soccer Robot Base of Northwestern Polytechnical University@Hawking for RoboCup competition. The code will be available by the end of this year.

  • Pytorch Implementation of Deeper and Wilder Siamese Network for Visual Tracking [Code]

    This is an unofficial Pytorch implementation of tracker SiamDW presented by the CVPR2019 paper Deeper and Wilder Siamese Network for Real-Time Visual Tracking. Particularly, I completed the train code for SiameRPN based tracker, optimized the framework for Siamese trackers and also conducted experiments on severl current popular datasets for comparison. The result has reached close performance as author presented in the paper.
